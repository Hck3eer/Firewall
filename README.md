\# VM Network Firewall System



A comprehensive firewall system with CLI and GUI interfaces that can block traffic across multiple VMs on the same network.



\## Features



\- \*\*Cross-VM Blocking\*\*: Block IPs/domains across all VMs on the network

\- \*\*Dual Interface\*\*: Both CLI and GUI interfaces

\- \*\*Real iptables Integration\*\*: Actual kernel-level blocking

\- \*\*Network Discovery\*\*: Automatically finds other VMs

\- \*\*Domain Resolution\*\*: Blocks all IPs associated with a domain

\- \*\*Rules Persistence\*\*: Saves rules to JSON/YAML files

\- \*\*Logging\*\*: Comprehensive activity logging
